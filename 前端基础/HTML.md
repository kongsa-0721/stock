### 一、布局标签

div , span 我们俗称盒子。

div

双标签，经典容器级别标签。作用是划分页面区域，辅助页面布局，将一些内容，布局相似的标签放到一个div盒子标签里进行统一管理。方便后期维护项目。

span

双标签，经典文本级别标签。在不影响整体布局的前提下，可以进行局部调整

### 二、表格

table 表格大框架

tr 表格内部的行

td 表格单元格 每一行中的单元格

table里嵌套tr tr里嵌套td

th 表头单元格 可以替换td 的位置，默认样式，文字居中，字体相对变大。

### 三、表单元素

可以让用户输入文字的一个小控件

1.form 表单 ，结构性标签，将表单元素收集到的数据提交给服务端，双闭合

属性：method : 设置提交数据的方式（post/get）

​           action ：设置收集到的表单的数据提交给谁的路径

2.文本框

**input ：表单元素 ， 单闭合标签，**

属性： **type** ：根据type属性值不同，创建不同的类型的表单元素。

type属性值：text 创建一个文本框

value ：默认值，默认显示的

3.密码框：可以通过input 标签获取到，只需将type 的属性值设置为password

4.单选按钮 也是通过input 标签获取到，只不过type的属性值为 radio

设置互斥方法，同时给这组单元按钮添加name值，name值必须相同

5.复选框 通过input标签获取 ，type属性值为checkbox

属性checked 属性值 checked 默认被选中

6.按钮 可以通过input 获取到三种不同类型的按钮

button 普通按钮

reset 重置按钮，单击这个按钮后，form 表单内到全部表单元素回复起始状态

submit 提交按钮 可以将form 标签内部收集到的 数据提交给服务器，全部元素恢复起始状态

7.文本域

文本框	<input type ="text”/>

文本域 textarea 双闭合文本级别标签，可以把他当成一个文字显示即可。

rows ，cols 这两个主要属性可以修改文本域的大小

rows  是显示的行数

cols   是设置文本域的宽度

可以设置默认文字显示，在内容部分当中插入即可

style = “resize  ：none”  让文本域不能拖拽进行放大缩小

8.下拉菜单

select ，option  双标签文本级

select 内可以嵌套option ，表示搭建下拉菜单的整体结构

option 表示下拉菜单当中每一个项

如果想让某一项作为默认选项，给option添加一个属性selected 值也为selected
